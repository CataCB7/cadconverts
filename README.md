# Website pentru HoReCa

Site premium static HTML/CSS/JS pentru prezentarea serviciilor de website-uri pentru hoteluri, pensiuni, cabane și restaurante.

## Fișiere
- index.html
- style.css
- script.js

## Ce trebuie modificat
În `index.html` și `script.js`, caută:
- `40700000000` și schimbă cu numărul tău de WhatsApp
- `contact@exemplu.ro` și schimbă cu emailul tău
- linkurile demo din secțiunea Exemple

## Notă
Exemplele și feedback-ul sunt prezentate ca demo/portofoliu, nu ca lucrări plătite pentru clienți.
